### ![small_red_triangle](https://github.githubassets.com/images/icons/emoji/unicode/1f53a.png) Instalação OpenJDK

***O OpenJDK (Kit de Desenvolvimento Java Aberto)** é uma  implementação gratuita e de código aberto da linguagem de  programação  Java.  A implementação está licenciada sob a GNU General  Public License  (GPL) com uma exceção de vinculação. **JDK = JRE + JVM***

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **1.** Abra o terminal e vamos verificar se temos o Java instalado:

```
java -version
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **2.** Para instalar o openJDK-11, digite no terminal: *A versão mais atual LTS é do Java 11, que terá seu suporte   estendido até Setembro de 2022. Este tipo de suporte iniciou no Java 8   que será mantido até 2023.*

```
sudo apt-get install openjdk-11-jdk
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **3.** Confirme se realmente foi instalado com sucesso:

```
java -version
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png)**4.** Vamos configurar o ambiente JAVA_HOME:

​	**4.1** Verificar o caminho da instalação do Java:

```
sudo update-alternatives --config java
```

​	**4.2** Copie o caminho que aparecerá no terminal, no meu caso:

```
/usr/lib/jvm/java-11-openjdk-amd64/bin/java
```

​	**4.3** Vamos editar o arquivo .bashrc:

```
sudo gedit ~/.bashrc
```

​	**4.4** Copie o código abaixo e cole no final do arquivo .bashrc.

​	*IMPORTANTE: cuidado para não alterar nada no arquivo além de apenas colar no final do mesmo o que vou te disponibilizar a  seguir.* 

​	*JAVA_HOME = aqui você coloca o caminho do tópico 4.2, tirando o /bin/java*

```
JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export JAVA_HOME
export PATH=$PATH:$JAVA_HOME
```

​	**4.3** Salve o arquivo

​	**4.4** Vamos conferir se a alteração ficou salva:

```
cat ~/.bashrc
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png)**5.**  Feche o terminal e abra novamente

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png)**6.** Vamos conferir mais uma vez se o Java está instalado na nossa máquina

```
java --version
```

*Créditos: [DevSuperior](https://www.youtube.com/watch?v=jARiy3DZdwg)*

### ![small_red_triangle](https://github.githubassets.com/images/icons/emoji/unicode/1f53a.png) Instalação Git 

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **1.** Abra o terminal (Ctrl + Alt + t) e vamos verificar se temos o git instalado:

```
git --version
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **2.** Execute o comando:

```
sudo apt-get install git-all
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png)**3.** Confirme novamente se o git realmente está instalado:

```
git --version
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **4.** Vamos começar as configurações iniciais:

​	**4.1** Cofigurar o nome de usuário

```
git config --global user.name "Seu nome"
```

​	**4.2** Configurar o endereço de e-mail: 	*É de suma importância que o ENDEREÇO DE E-MAIL SEJA O MESMO DO GITHUB afim de evitar conflitos!*

```
git config --global user.email seuemail@email.br
```

​	**4.3** Vamos conferir a lista de configurações:

```
git config --list
```

![small_orange_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f538.png) **5.** Pronto, git instalado e configurado com sucesso!

### ![small_red_triangle](https://github.githubassets.com/images/icons/emoji/unicode/1f53a.png) Instalação JDK Zulu

Aqui no windows, vamos fazer o download do **OpenJDK Zulu**. *As compilações do Azul Zulu do OpenJDK são compilações de código  aberto, testadas pelo TCK e certificadas do OpenJDK. O Zulu Blue está  disponível para uma ampla variedade de plataformas de hardware e  sistemas operacionais. A documentação do Azul Zulu inclui notas de  lançamento, um guia de instalação e licenças de terceiros.*

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **1.** Entre no [**SITE AZUL**](https://www.azul.com/downloads/?package=jdk)

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **2.** Faça o download do arquivo .zip do JDK 11.0.11+9. No meu caso, o x86 64-bit

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **3.** Vá no drive C://Arquivo de Programas

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **4.** Caso não tenha um diretório com o nome Java, crie

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **5.**  Entre neste diretório e descompacte o download do zip JDK Zulu 11.0.11+9 neste diretório

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **6.** Vamos configurar o ambiente JAVA_HOME:

​	**6.1**  Menu iniciar -> Editar as varáveis de ambiente do sistema

​	**6.2** Irá abrir a janela Propriedades do Sistema, escolha a aba Avançado, em seguida clique em variáveis de Ambiente

​	**6.3** Na janela Variáveis de Ambiente,  crie um novo Variáveis do sistema

​	**6.4** Abrirá uma jabela: Nova Variável de Sistema.

​	**6.5** Nome da variável: JAVA_HOME

​	**6.6** Valor da variável: em seguida OK. 	*O valor da variável é o caminho do diretório que você descompactou o zip JDK Zulu 11.0.11+9 no passo 5* 

​	**6.7** Na mesma janela Variáveis do Sistema, localize a variável Path, selecione e clique a opção Editar...

​	**6.8** Clique na opção Novo e cole o mesmo caminho do passo 5 acrescentando \bin

​	**6.9** Continue com o path selecionado e clique na opção Mover para Cima até chegar no topo

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **7.** Pronto, finalizada a configuração. Próximo passo é conferir se está instalado tudo certinho

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **8.** Abra o Prompt de Comando: Menu iniciar -> cmd

![small_blue_diamond](https://github.githubassets.com/images/icons/emoji/unicode/1f539.png) **9.** Vamos conferir mais uma vez se o Java está instalado na nossa máquina

```
java -version
```
